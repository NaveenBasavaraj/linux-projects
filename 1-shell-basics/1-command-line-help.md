# Command Line Help

1. whatis:
```
whatis date
> date (1)             - print or set the system date and time
```

2. man command for more detail description with options:
```
man date
```

3. --help to get help:
```
date --help
```

4. apropos:
```
apropos modir (check if such command exists)
```
